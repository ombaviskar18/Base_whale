```

#### 🟡 RECOMMENDED (For full functionality):
```

#### 🟢 OPTIONAL (For advanced features):
```

TELEGRAM_BOT_TOKEN=your_telegram_bot_token
ENCRYPTION_KEY=your_32_character_encryption_key
```
